# Estudio Juridico Dr. Julio Leiva

Sitio web estatico desarrollado con HTML, CSS y JavaScript para publicar en GitHub Pages.

## Contenido

- Presentacion profesional del estudio juridico
- Enfoque en derecho, tecnologia y pericia informatica
- Diseno responsive para desktop y celular
- Formulario de contacto por `mailto:` a `drjulioleiva@gmail.com`

## Publicacion en GitHub Pages

1. Crear un repositorio en GitHub, por ejemplo `estudio-juridico`.
2. Subir el contenido de este directorio.
3. En GitHub:
   - abrir `Settings`
   - ir a `Pages`
   - en `Build and deployment`, elegir `Deploy from a branch`
   - seleccionar la rama `main`
   - elegir `/ (root)`
4. Guardar y esperar la publicacion.
